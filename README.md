# Boat Overloading Detection System

## Overview
This project aims to develop a system for detecting overloading in boats, ensuring safety and compliance with regulations.

## Features
- Real-time monitoring of boat weight.
- Alerts when weight exceeds safety limits.
- User-friendly interface.

## Tech Stack
| Technology       | Description            |
|------------------|------------------------|
| Python           | Backend logic          |
| Flask            | Web framework          |
| JavaScript       | Frontend development    |
| PostgreSQL       | Database management    |

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/mwakyomax/Boat_overloading_detection_system.git
   ```
2. Navigate into the project directory:
   ```bash
   cd Boat_overloading_detection_system
   ```
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
Run the application using:
```bash
python app.py
```
